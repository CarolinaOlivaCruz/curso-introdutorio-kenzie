O que são as variáveis? Variáveis são espaços na memória em que colocamos valores para podermos trabalhar com eles no nosso código. Ao declarar uma variável, podemos gravar e ler esses valores diretamente dela a partir do nome que definimos para a variável. String - Uma string é um texto.
